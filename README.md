# Café Associatif

Site vitrine pour un café associatif chaleureux et convivial. Ce projet a été développé dans le cadre d'un TD collaboratif utilisant Git et GitHub.

## Description

Le site se compose de 3 pages :

- **Accueil** : Présentation du café associatif
- **Menu** : Carte des boissons et pâtisseries
- **Contact** : Formulaire de contact et coordonnées

## Technologies utilisées

- HTML5
- CSS3
- Git & GitHub
